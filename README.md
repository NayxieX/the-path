# The Path

Консольна рольова гра на Java — навчальний проєкт з ООП.

Герой прокидається у в'язниці без жодних спогадів і проходить п'ять розділів історії: темницю, ліс, село, замок і фортецю. Кожен діалоговий вибір впливає на репутацію, а репутація змінює тон, репліки й навіть подальші сюжетні гілки. У фіналі гравця чекає одна з трьох кінцівок залежно від того, ким він став за час подорожі.

## Особливості

- П'ять розділів з розгалуженим сюжетом і наслідками, що переносяться з розділу в розділ (рішення в лісі впливають на події в замку, рішення в селі — на те, що відбувається далі, і так до фіналу).
- Система репутації (Герой / Нейтральний / Антагоніст), яка змінює репліки NPC, поведінку ворогів і фінальну кінцівку.
- Бойова система з кількома варіантами дій (вбити, знешкодити, залякати, обшукати), кожен зі своєю ціною для репутації.
- Фінальна зустріч з босом — не звичайний бій, а діалог: гравець обирає питання Морлоку замість ударів, і кожна відповідь розкриває частину його історії.
- Наскрізні сюжетні мотиви й деталі, що пов'язують усі п'ять розділів в одну історію.

## Технології

Java, консольний інтерфейс, ООП (абстрактні класи, інтерфейси, наслідування, enum, record, кастомні винятки).

## Структура проекту
src/rpg/

├── Main.java

├── entities/    — Character, Hero, NPC, Enemy, Morlock

├── chapters/    — Chapter1...Chapter5

├── mechanics/   — CombatSystem, DialogSystem, ReputationSystem

├── enums/       — ReputationLevel, CombatAction, EndingType

├── interfaces/  — Chapter, ReputationReactive

├── exceptions/  — InvalidChoiceException

└── util/        — ConsoleUI
